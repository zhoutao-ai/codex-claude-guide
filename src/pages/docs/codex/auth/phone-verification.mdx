## 概述

手机号验证是 Codex 用户（尤其是中国大陆用户）遇到的最大门槛。OpenAI 在创建首个 API Key 或新设备登录时会强制要求手机号验证，而 +86 号码无法使用。本指南提供完整的解决方案。

## 问题根源

OpenAI 的手机号验证系统有以下限制：

| 限制 | 说明 |
|---|---|
| 不支持 +86 | 中国大陆号码完全排除 |
| 拒绝 VoIP | Google Voice、TextNow 等虚拟号码被识别 |
| 号码绑定上限 | 每号最多绑定 2-3 个账号 |
| 风控检测 | IP 与号码国家不一致时可能拒绝 |
| 黑名单机制 | 被大量使用的公共号码会被拉黑 |

## 解决方案总览

| 方案 | 稳定性 | 成本 | 难度 | 推荐度 |
|---|---|---|---|---|
| 海外实体 SIM 卡 | ★★★★★ | 中 | 低 | ★★★★★ |
| 长期独占云号码 | ★★★★ | 低-中 | 低 | ★★★★ |
| ChatGPT Plus 订阅 | ★★★★ | 高 | 低 | ★★★ |
| 自建短信转发 | ★★★★ | 中 | 高 | ★★★ |

## 方案一：海外实体 SIM 卡（最推荐）

### 适合人群

- 长期使用 OpenAI/Codex 的开发者
- 有渠道购买海外 SIM 卡的用户
- 需要最高稳定性的用户

### 推荐卡种

| 卡种 | 国家 | 月费 | 优势 |
|---|---|---|---|
| Ultra Mobile PayGo | 美国 | $3/月 | 可在国内漫游收短信 |
| Lycamobile | 美国/英国 | £0-10/月 | 多国可选 |
| Knowroaming | 美国 | $5/月 | 全球漫游 |
| giffgaff | 英国 | £0-6/月 | 免月费可保号 |

### 操作步骤

1. 在淘宝/闲鱼/官网购买 SIM 卡
2. 按说明激活（部分卡需在美国 IP 下激活）
3. 确保能在中国接收国际短信（测试给该号码发一条短信）
4. 在 OpenAI 验证页面输入号码
5. 等待接收验证码短信
6. 输入验证码完成验证

<div class="alert alert-tip">
<strong>保号技巧：</strong>大部分海外预付费卡需要定期发送短信或充值来保持活跃。建议每 3 个月发送一条短信或充值最低金额，防止号码被回收。
</div>

## 方案二：长期独占云号码

### 适合人群

- 没有渠道购买实体卡的用户
- 追求低成本的用户

### 推荐平台

| 平台 | 类型 | 特点 | 价格 |
|---|---|---|---|
| DuoPlus | 长期独占 | 支持支付宝，多国号码 | $2-5/月 |
| SMS-Activate | 长期租用 | 老牌平台，号码多 | $1-10/月 |
| 5SIM | 长期租用 | 界面友好 | $1-5/月 |

<div class="alert alert-danger">
<strong>避免使用：</strong>
<ul>
<li>免费接码网站（号码被反复使用，100% 失败）</li>
<li>一次性接码服务（验证完就失效，后续无法重新验证）</li>
<li>淘宝低价"代注册"服务（可能用你的信息注册后转卖）</li>
</ul>
</div>

### 操作步骤

1. 在平台注册并充值
2. 选择**长期独占**（不是一次性）的美国/英国号码
3. 确认号码未被使用过
4. 在 OpenAI 验证页面输入号码
5. 在平台查看收到的验证码
6. 输入验证码完成验证
7. **保留号码订阅**，以备后续重新验证

## 方案三：ChatGPT Plus 订阅

如果你订阅了 ChatGPT Plus ($20/月)，可以在 [chatgpt.com/codex](https://chatgpt.com/codex) 直接使用 Codex，无需单独创建 API Key 和验证手机号。

ChatGPT Plus 订阅时需要完成手机号验证，但可以通过 Google/Apple 账号关联注册来跳过验证。

### 操作步骤

1. 使用 Google 账号或 Apple ID 登录 [chatgpt.com](https://chatgpt.com)
2. 如果提示手机号验证，尝试使用 Google 账号关联（有时可跳过）
3. 订阅 ChatGPT Plus
4. 在 Codex CLI 中使用 ChatGPT 账号登录

## 防风控核心原则

### 1. IP 一致性

VPN/代理的 IP 国家**必须**与手机号归属国一致：

| 号码国家 | IP 要求 | 示例 |
|---|---|---|
| 美国 (+1) | 美国 IP | ✓ |
| 英国 (+44) | 英国 IP | ✓ |
| 美国 (+1) | 日本 IP | ✗（风控风险） |

### 2. 环境干净

- 使用浏览器无痕模式
- 清除所有 Cookie 和缓存
- 不要使用安装了过多插件的非无痕窗口
- 确保之前没有在同一浏览器多次尝试失败

### 3. 操作间隔

- 如果验证失败，**等待 24 小时**再重试
- 不要频繁点击"发送验证码"按钮
- 更换号码后也要等待一段时间再验证

### 4. 一次到位

- 确保号码可用后再输入
- 验证码有效期通常为 5-10 分钟
- 收到后立即输入，不要拖延

## 验证失败后的处理

| 失败提示 | 含义 | 处理方法 |
|---|---|---|
| Unsupported phone line type | VoIP 被识别 | 换实体卡或独占号 |
| Number already used | 号码配额用完 | 换全新号码 |
| Temp blocked | 试错太多被锁 | 等 24 小时 |
| No code received | 验证码没收到 | 检查号码是否支持短信 |

## 常见问题

### Q: 验证成功后还需要保留号码吗？

建议保留。如果 OpenAI 后续要求重新验证（新设备登录等），需要使用同一号码。如果号码丢失，可能导致账号无法恢复。

### Q: 可以用朋友的海外号码吗？

可以，但该号码会绑定到你的账号。如果朋友也需要注册 OpenAI，号码配额可能不够。

### Q: 有没有办法完全跳过手机号验证？

目前没有合法的方式跳过。唯一"绕过"的方法是使用 ChatGPT Plus 订阅（注册时可能通过 Google 账号跳过验证）。
