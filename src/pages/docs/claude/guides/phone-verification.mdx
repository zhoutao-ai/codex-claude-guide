## 概述

手机号验证是中国大陆用户注册 Claude 时遇到的最大障碍。Anthropic 的验证系统比 OpenAI 更严格，对号码类型、IP 环境、操作频率都有严格的风控。本指南提供完整的解决方案。

## 问题全景

| 问题 | 原因 | 解决方向 |
|---|---|---|
| +86 号码不可用 | 中国大陆被排除 | 使用海外号码 |
| VoIP 被拒 | 虚拟号被识别 | 使用实体 SIM 卡 |
| 验证码收不到 | 区号不在服务区 | 使用美/英号码 |
| Number already used | 号码绑定达上限 | 换全新号码 |
| Temp blocked | 尝试过多被锁 | 等待 24 小时 |

## 号码方案对比

### 方案一：海外实体 SIM 卡（最推荐）

| 卡种 | 国家 | 月费 | 漫游收短信 | 购买渠道 |
|---|---|---|---|---|
| Ultra Mobile PayGo | 美国 | $3 | ✓ | 淘宝 |
| Lycamobile | 美国/英国 | $2-10 | ✓ | 淘宝 |
| giffgaff | 英国 | £0-6 | ⚠ 不稳定 | 淘宝 |
| T-Mobile 预付费 | 美国 | $3-10 | ✗ 不支持 | 需在美国激活 |

**操作步骤：**

1. 在淘宝搜索 "Ultra Mobile" 或 "美国 SIM 卡"
2. 购买后按说明激活（需在美国 IP 下激活部分卡种）
3. 将 SIM 卡插入手机，开启国际漫游
4. 测试收发短信功能
5. 在 Claude 注册页面输入号码
6. 接收验证码并完成验证

**优点：** 最稳定，几乎 100% 成功
**缺点：** 需要购买实体卡，有月费成本

### 方案二：长期独占云号码

| 平台 | 类型 | 独占号码价格 | 特点 |
|---|---|---|---|
| DuoPlus | 长期独占 | $2-5/月 | 支持支付宝 |
| SMS-Activate | 长期租用 | $1-10/月 | 老牌平台 |
| 5SIM | 长期租用 | $1-5/月 | 界面友好 |

**操作步骤：**

1. 在平台注册并充值
2. 选择**长期独占**（非一次性）美国号码
3. 确认号码类型为 "Mobile"（非 VoIP）
4. 在 Claude 注册页面输入号码
5. 在平台查看收到的验证码
6. **保留订阅**，以备后续重新验证

<div class="alert alert-danger">
<strong>避免：</strong>免费接码网站（号码被反复使用，100% 失败）、一次性接码（验证完就失效）。
</div>

### 方案三：Google/Apple 账号注册

Claude 支持第三方账号登录，有时可以跳过手机号验证：

1. 在 claude.ai 选择 "Continue with Google"
2. 使用已登录的 Google 账号授权
3. 部分用户反馈无需手机号验证

**成功率：** 约 30-50%，不稳定，但值得一试。

### 方案四：eSIM 方案

如果你有支持 eSIM 的手机：

1. 购买海外 eSIM 方案（如 Airalo、GigSky）
2. 激活 eSIM 获得海外号码
3. 使用该号码进行验证

**注意：** 部分 eSIM 方案仅提供数据，不提供号码。购买前确认包含电话号码。

## 防风控核心原则

### 1. IP 一致性

```
VPN IP 国家 = 手机号归属国
```

| 号码国家 | IP 要求 | 状态 |
|---|---|---|
| 美国 (+1) | 美国 IP | ✓ |
| 英国 (+44) | 英国 IP | ✓ |
| 美国 (+1) | 日本 IP | ✗ 风控 |

### 2. 环境干净

- 使用 Chrome/Edge **无痕模式**
- 清除所有 Cookie 和缓存
- 不要使用之前注册失败的浏览器配置
- 关闭浏览器中可能暴露真实 IP 的插件

### 3. 时区一致

浏览器时区应与 IP 所在时区一致：

- 美国 IP → 时区设为 UTC-5 到 UTC-8
- 英国 IP → 时区设为 UTC+0

在 Chrome 地址栏输入 `chrome://settings/languages` 修改语言和时区。

### 4. 操作节奏

| 操作 | 建议间隔 |
|---|---|
| 两次验证码请求 | ≥ 60 秒 |
| 验证失败后重试 | ≥ 1 小时 |
| 更换号码后验证 | ≥ 30 分钟 |
| 被临时封锁后 | ≥ 24 小时 |

## 各错误场景详细处理

### 场景一：Unsupported phone line type

**原因：** 使用了 VoIP 号码

**处理：**
1. 停止使用 VoIP 号码（Google Voice、TextNow 等）
2. 获取实体 SIM 卡
3. 等待 1 小时后用新号码验证

详见 [Unsupported phone line type 详解](/docs/claude/errors/unsupported-phone)

### 场景二：验证码收不到

**原因：** 区号不在服务区或运营商拦截

**处理：**
1. 确认号码区号在支持列表（+1、+44 等）
2. 检查手机漫游设置
3. 测试号码是否能正常收短信
4. 更换为美国号码

详见 [验证码收不到详解](/docs/claude/errors/no-code-received)

### 场景三：Number already used too many times

**原因：** 号码已绑定太多 Claude 账号

**处理：**
1. 更换全新号码
2. 确保新号码从未用于 Claude
3. 使用全新购买的实体 SIM 卡

详见 [Number already used 详解](/docs/claude/errors/number-used)

### 场景四：phone_number_temp_blocked

**原因：** 尝试过多被临时锁定

**处理：**
1. 停止所有尝试
2. 等待 24 小时
3. 更换号码和 IP 环境
4. 重新验证

详见 [phone_number_temp_blocked 详解](/docs/claude/errors/temp-blocked)

## 验证成功后的保号建议

1. **保留号码订阅**：不要取消云号码或停用 SIM 卡
2. **定期活动**：每 1-3 个月用该号码收发一条短信，保持活跃
3. **记录号码信息**：保存号码、平台账号、充值记录
4. **不要在其他账号使用同一号码**：一号一账号

## 常见问题

### Q: Claude 的验证比 OpenAI 更难吗？

是的。Anthropic 对号码类型的检测更严格，在 OpenAI 上能通过的 VoIP 号码在 Claude 上几乎都会被拒。

### Q: 验证后可以更换绑定的号码吗？

Claude 目前不提供自助更换号码的功能。如果号码丢失，可能需要联系客服。

### Q: 使用朋友的海外号码可以吗？

可以，但该号码会绑定到你的账号。如果朋友也需要注册 Claude，号码配额可能不够。
